p# Images for CurePocket Slide Deck

このフォルダには、スライドで使用する画像を配置してください。

## 必要な画像

### 1. ロゴ
- **ファイル名**: `logo.png`
- **用途**: タイトルスライド
- **推奨サイズ**: 200x200px 程度（透過PNG推奨）
- **説明**: CurePocketのロゴ（ハート×ポケットデザイン）

### 2. プロフィール写真
- **ファイル名**: `profile-1.png`, `profile-2.png` (必要に応じて `profile-3.png`)
- **用途**: タイトルスライドの自己紹介セクション
- **推奨サイズ**: 120x120px（正方形、円形にトリミングされます）
- **説明**: Shizukuさんのプロフィール写真

### 3. UI モックアップ
- **ファイル名**: `ui-mockup.png`
- **用途**: ソリューション概要スライド
- **推奨サイズ**: 400x800px 程度（スマートフォン画面風）
- **説明**: CurePocketのUIスクリーンショットまたはモックアップ

### 4. テクノロジーロゴ
- **ファイル名**:
  - `sui-logo.png`
  - `walrus-logo.png`
  - `seal-logo.png`
- **用途**: アーキテクチャダイアグラムスライド
- **推奨サイズ**: 高さ60px程度（幅は自動調整）
- **説明**: 各テクノロジーの公式ロゴ

## 画像の配置方法

1. 上記のファイル名で画像をこのフォルダに配置してください
2. PNG形式を推奨（透過背景が必要な場合は必須）
3. ファイル名は必ず上記のものと一致させてください

## 注意事項

- 画像が見つからない場合、スライドには画像のプレースホルダーが表示されます
- すべての画像は自動的にレスポンシブ対応されます
- ロゴやプロフィール写真には自動的にシャドウ効果が適用されます
